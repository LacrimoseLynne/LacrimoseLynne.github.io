Hexo框架学习笔记
2019年12月06日 581 字 大概 2 分钟 学习笔记

Hexo常用指令

npm install hexo -g ————– 安装Hexo

npm update hexo -g ————– 升级

hexo init 文件名 ————– 初始化博客 文件名指定了博客存放的目录

hexo n(new) “文章名” ———– 新建一篇文章

hexo g(generate) ———- 生成静态网页

hexo s(server) ———– 可以在本地预览网站效果(Hexo自带浏览器 地址: https://localhost:4000)

hexo d(deploy) ———– 部署

hexo server #Hexo 会监视文件变动并自动更新，无须重启服务器

hexo server -s ————- 静态模式

hexo server -p 5000 ———– 更改端口

hexo server -i 192.168.1.1 ———— 自定义 IP

hexo clean ————- 清除缓存，若是网页正常情况下可以忽略这条命令
